# 🚀 TEXT-TO-SQL

A web-based application that converts natural language (English) into SQL queries
and executes them on user-created databases.

Built using **FastAPI + SQLite + HTML/CSS/JS**.

---

## 📌 Features

✅ Create custom databases and tables  
✅ Insert data using simple English  
✅ Fetch records with filters  
✅ Delete and count records  
✅ Multiple database support  
✅ Dynamic schema handling  
✅ Modern UI  
✅ No external AI API needed  

---

## 🛠 Tech Stack

- Backend: FastAPI (Python)
- Database: SQLite
- Frontend: HTML, CSS, JavaScript
- API: REST
- Server: Uvicorn
